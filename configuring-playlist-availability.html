Playlist availability details can be used to restrict access to {{ site.product_short }} playlists based upon date, locations and devices. To view and edit the availability details for a {{ site.product_short }} playlist, follow these steps:

  1. Open the Media module.
  2. In the panel navigation, click OTT playlists.
  3. Click on a playlist title to open the playlist details page.
  4. Click Edit Settings.
  5. Click the Availability tab.

The availability settings are controlled using rule sets. All playlists must have at least one rule set. Rule sets are created to control access to a video based upon:

  • Start/end dates
  • Locations
  • Devices
For information on creating and managing rule sets, see Creating and Managing Rule Sets.
