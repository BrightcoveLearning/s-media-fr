Pour afficher et modifier les détails d'une vidéo, ouvrez le module Médias. Dans le panneau de navigation, cliquez sur Toutes les vidéos, puis sur le lien du titre d'une vidéo. La page des détails de la vidéo s'ouvre.

Remarque: Pour modifier les détails de plusieurs vidéos en même temps, utilisez la fonction Quick Edit.

Les boutons en haut de la page des détails de la vidéo permettent de:

  • Capturez de nouvelles affiches et images miniatures
  • Ajoutez la vidéo à un dossier ou ajoutez-la à une liste de lecteur
  • Publier une vidéo
  • Désactiver/activer les vidéos
  • Supprimer la vidéo

Si la vidéo est activée, le bouton Désactiver peut être utilisé pour désactiver la vidéo. Si la vidéo est désactivée, le bouton Activer peut être utilisé pour activer la vidéo. Les vidéos désactivées ne peuvent pas être lues dans un lecteur Brightcove.

Le lecteur de prévisualisation en haut de la page peut être utilisé pour visionner la vidéo. La chronologie des points de repère affichera tous les points de repère qui ont été ajoutés à la vidéo.

Les onglets situés sous la chronologie du point de repère organisent les détails de la vidéo en sections:

  • Aperçu
    • Informations sur la vidéo
    • Informations sur le type vidéo
    • Casting
    • Catégorisation
    • Champs personnalisés
  • Disponibilité
  • Vidéo et pistes
    • Pistes de texte
    • Fichiers vidéo
    • Partage de médias
    • Pistes audio
  • Images

Pour modifier les détails de la vidéo, cliquez sur le bouton Modifier dans une section ou double-cliquez dans une section.

Champs obligatoires

Votre compte Brightcove {{site.product_short}} peut être configuré avec des champs de métadonnées obligatoires. Si une vidéo est inactive, vous pouvez modifier et enregistrer librement ses métadonnées sans fournir toutes les métadonnées requises. En outre, les vidéos ne peuvent pas être activées tant que toutes les métadonnées requises n'ont pas été saisies. Si une vidéo est déjà active, vous ne pouvez enregistrer aucune modification dans la section que vous modifiez, sauf si vous fournissez toutes les métadonnées requises dans cette section ou si vous désactivez la vidéo. Pour plus d'informations sur l'utilisation des champs obligatoires, consultez la section Définition des champs obligatoires pour les vidéos.

Informations sur la vidéo

La section Informations sur la vidéo peut être utilisée pour afficher et modifier les détails de base de la vidéo.

Pour modifier les détails, cliquez sur le bouton Modifier ou double-cliquez dans la section. Les informations suivantes peuvent être modifiées:

  • Nom: 255 caractères maximum
  • ID: identifiant vidéo attribué par le système (unique sur tous les comptes Brightcove)
  • Brève description - Pas plus de 250 caractères
  • Tags: un ou plusieurs tags, séparés par des virgules ; utilisés pour ajouter des vidéos à des listes de lecteur intelligentes
    • Les balises peuvent contenir jusqu'à 128 caractères. Si l'une des balises comporte plus de 128 caractères, une erreur s'affiche lorsque vous tentez d'enregistrer les modifications.
    • Les balises ne peuvent pas contenir de virgules. Par exemple, une étiquette toit, grenier + cour avant-fr/ n'est pas prise en charge.
    • Une vidéo ne peut pas contenir plus de 1200 balises
    • Une balise qui ne diffère que par la casse (minuscules, majuscules) est traitée comme une étiquette identique et la casse correspond automatiquement à celle de la balise d'origine
    • Les problèmes de respect de la casse doivent être traités en externe à Brightcove {{site.product_short}}, par exemple en utilisant des méthodes JavaScript pour modifier les chaînes
    • Après avoir saisi une balise, appuyez sur la virgule ou sur Entrée. Les balises qui ont été ajoutées s'afficheront dans une boîte colorée.
  • ID de référence: identifiant unique attribué par un éditeur à une vidéo ou à une liste de lecteur. Limité à 150 caractères maximum. L'ID de référence est distinct de l'identifiant de la vidéo ou de la playlist, qui sont des ID uniques générés par Brightcove. L'ID de référence permet la coordination entre Video Cloud et d'autres services tels que le système de gestion de contenu d'un éditeur. Notez que vous ne pouvez pas modifier l'ID de référence par lots, car deux vidéos ne peuvent pas avoir le même ID de référence. Les ID de référence doivent toujours être uniques. La suppression d'une vidéo permettra de réutiliser un identifiant de référence précédemment attribué.
  • Projection vidéo: indique comment la vidéo a été enregistrée
    • Standard - Par défaut. Enregistrement vidéo régulier.
    • 360 degrés - Cochez cette case pour indiquer que la vidéo est une vidéo à 360 degrés. Un badge 360 apparaîtra dans la miniature de la vidéo dans le module Médias. Pour plus d'informations sur l'utilisation de vidéos à 360 degrés, consultez la section Utilisation de vidéos à 360 degrés.
  • Activer la lecteur hors connexion: lorsque cette case est cochée, la vidéo peut être enregistrée localement et visionnée hors ligne lors de l'utilisation d'applications créées à l'aide des kits SDK de l'appareil Cette option n'apparaît que si le compte est activé pour la lecteur hors ligne.
  • URL du lien associé: URL Web standard commençant par http://ou https://limitée à 255 caractères
  • Related Link Text - Texte du lien associé limité à 255 caractères
  • Description longue - Pas plus de 5 000 caractères
  • Publicité
    • Gratuit - Les publicités ne seront pas diffusées lors de la lecteur de la vidéo
    • Publicité supportée - Les publicités peuvent être diffusées lors de la lecteur d'une vidéo
      Remarque: Si votre compte n'est pas activé pour la publicité et que vous définissez la propriété Publicité sur Ad Supported, vous recevrez un message d'erreur lors de l'enregistrement des modifications.
  • Clés publicitaires: transmises dans le cadre de l'URL du serveur publicitaire. Par exemple, pour transmettre section=blog et AnotherKey=Value1, vous devez entrer SECTION=BLOG&AnotherKey=value1. Consultez l'aide de DoubleClick for Publishers pour plus d'informations sur l'ajout de paires clé-valeur. Lors de l'utilisation de clés publicitaires, les macros publicitaires doivent être ajoutées à l'URL du serveur publicitaire (configurée dans le cadre des propriétés publicitaires du lecteur). Les macros publicitaires prennent en charge les variables de l'URL du serveur pour lesquelles le plug-in IMA3 remplacera les valeurs appropriées. Pour plus d'informations, consultez la documentation destinée aux développeurs IMA3.
  • Partage: indique si la vidéo a été partagée sur un autre compte. Pour plus d'informations, consultez la section Partage de vidéos avec d'autres comptes.

Cliquez sur Enregistrer pour enregistrer les modifications.

Informations sur le type vidéo

La section Informations sur le type de vidéo fournit des détails vidéo supplémentaires en fonction du type de vidéo sélectionné. Les types de vidéos suivants sont disponibles:

  • Film: indique que la vidéo est un film. Lorsqu'une nouvelle vidéo est téléchargée, le type de vidéo est défini sur Film.
  • Episode - Indique que la vidéo est un épisode d'une saison. Le panneau Informations sur le type de vidéo affiche le numéro de l'épisode et la saison à laquelle la vidéo est attribuée.
  • Bande-annonce: indique que la vidéo est utilisée comme bande-annonce. Les vidéos ajoutées en tant que bande-annonce d'un film ou d'un épisode verront automatiquement leur type de vidéo remplacé par Bande-annonce.

Si le type de vidéo est Film ou Épisode, des champs vidéo supplémentaires peuvent être configurés:

  • Bande-annonce - Cliquez sur+Ajouter une bande-annonce pour sélectionner la bande annonce du film ou de l'épisode
  • Classement par âge - Sélectionnez la (les) classification (s) par âge pour les pays listés
  • Numéro de l'épisode - Entrez le numéro de l'épisode (épisodes uniquement)
  • Saison: saison à laquelle l'épisode a été associé (épisodes uniquement)

Casting

La section Cast peut être utilisée pour ajouter des informations de diffusion pour une vidéo. Tous les champs du panneau Fonte prennent en charge la recherche par saisie anticipée. Au fur et à mesure que vous tapez, les entrées correspondantes précédentes apparaissent et vous pouvez cliquer dessus pour les ajouter à la sélection.

Catégorisation

La section Catégorisation peut être utilisée pour configurer les détails suivants:

  • Genres - Les genres de la vidéo
  • Listes de lecteur manuelles: listes de lecteur auxquelles cette vidéo appartient. Pour plus d'informations, consultez la section Création et gestion de listes de lecteur.
  • Dossier: dossier auquel la vidéo est attribuée. Consultez Organisation des vidéos à l'aide de dossiers pour plus d'informations.

Champs personnalisés

La section Champs personnalisés peut être utilisée pour afficher et modifier le champ personnalisé de la vidéo. Pour modifier les valeurs des champs personnalisés, cliquez sur le bouton Modifier ou double-cliquez dans la section. Un astérisque (*) en regard du nom du champ indique que le champ est obligatoire.

Remarque: Pour plus d'informations sur l'ajout de champs personnalisés à un compte, consultez Création de champs de métadonnées personnalisés.

Disponibilité

L'onglet Disponibilité affiche les paramètres de disponibilité des vidéos. Les paramètres de disponibilité varient en fonction du type de vidéo sélectionné. Les paramètres de disponibilité ne sont pas disponibles pour les remorques. Pour plus d'informations sur la configuration des paramètres de disponibilité, consultez Configuration de la disponibilité de la vidéo.

Pistes de texte

La section Pistes de texte peut être utilisée pour ajouter des pistes de texte à la vidéo. Pour plus d'informations sur la configuration des pistes de texte, consultez Ajout de sous-titres aux vidéos.

Fichiers vidéo

La section Fichiers vidéo affiche le nom du fichier source, la technologie de transcodage utilisée et des détails sur les rendus vidéo créés. Il permet également de:

  • Remplacer le fichier source vidéo
  • Retranscoder la vidéo

Pour télécharger un format associé, cliquez avec le bouton droit de la souris sur le lien du format associé. Notez que seuls les rendus MP4 peuvent être téléchargés.

Remarque: Un commutateur à bascule DRM peut également apparaître dans la section si le compte a été activé pour les DRM. Pour plus d'informations sur les DRM, consultez la section Ingestion de contenu avec DRM.

Partage de médias

La section Partage de médias répertorie les comptes affiliés sur lesquels la vidéo a été partagée (le cas échéant). Pour plus d'informations sur le partage de médias, consultez la section Partage de vidéos avec d'autres comptes.

Pistes audio

La section Pistes audio affiche des informations sur les rendus audio disponibles. Pour plus d'informations sur l'ajout de pistes audio supplémentaires à une vidéo, consultez Ajout de pistes audio aux vidéos.

Remarque: La section Pistes audio ne s'affiche pas si la vidéo a été transcodée à l'aide d'un profil d'acquisition hérité ou si le compte n'est pas activé pour plusieurs pistes audio.

Images

L'onglet Images affiche les images de la vidéo. Les images d'affiches et de vignettes sont créées automatiquement en prenant un instantané de la vidéo source à mi-chemin. Si vous n'aimez pas les images par défaut qui ont été capturées, vous pouvez capturer de nouvelles images ou télécharger d'autres images pour les utiliser comme affiche et vignette. Les images paysage, portrait et ultra-large doivent être téléchargées manuellement. Passez la souris sur n'importe quelle image et cliquez sur l'icône d'affichage () pour l'afficher dans un nouvel onglet du navigateur.
