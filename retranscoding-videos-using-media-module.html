Brightcove se réserve le droit de mettre à jour les profils d'acquisition par défaut associés aux comptes {{site.product_short}}. Vous pouvez également avoir besoin de mettre à jour vos propres profils d'acquisition personnalisés. Lorsque les profils d'acquisition changent, vous pouvez souhaiter que les vidéos existantes de votre compte soient retranscodées afin qu'elles aient des rendus correspondant aux nouveaux paramètres de profil d'ingestion.

Remarque: La possibilité de retranscoder des vidéos existantes n'est disponible que pour les comptes Professional et Enterprise. Si cette fonctionnalité vous intéresse, veuillez contacter votre Account Manager.

Les vidéos peuvent être retranscodées à l'aide du module Media ou de l'API Dynamic Ingest. L'utilisation de l'API Dynamic Ingest peut être une meilleure solution lorsque vous avez un grand nombre de vidéos qui nécessitent un retranscodage. Pour retranscoder des vidéos à l'aide de l'API Dynamic Ingest, consultez le document destiné aux développeurs Dynamic Ingest API Overview.

Remarque: Vous ne pouvez pas retranscoder les vidéos qui ont été partagées sur votre compte.
Remarque: Seules les vidéos qui ont été téléchargées à l'aide du module Upload ou de l'API Dynamic Ingest peuvent être retranscodées. Si la vidéo ne peut pas être retranscodée, le retranscodage échouera.
Remarque: les vidéos ne peuvent être retranscodées que si le masque vidéo a été archivé lorsque la vidéo a été ajoutée au compte. Les vidéos plus anciennes peuvent ne pas avoir de master, selon le profil d'ingestion utilisé.

Étapes pour retranscoder une vidéo

Suivez ces étapes pour remplacer les rendus existants d'une vidéo.

  1. Ouvrez le module Médias.
  2. Cliquez sur la vidéo que vous souhaitez retranscoder.
    3. {% if site.product_short == Beacon-fr/ ou site.product_short == CorpTV-fr/ %}
  3. Cliquez sur l'onglet Vidéo et pistes.
    4. {% endif %}
  4. Dans la section Fichiers vidéo, cliquez sur Retranscoder.
  5. Sélectionnez un profil d'acquisition à utiliser.
  6. Cochez la case Conserver les images de vignettes et d'affiches existantes pour conserver les images existantes. Si vous décochez cette case, de nouvelles images seront générées en fonction des paramètres d'image du profil d'acquisition sélectionné.
    Notez que si vous cochez la case, les images existantes seront conservées, mais les URL des images changeront.
    Remarque: Si le compte est activé pour le filigrane judiciaire, il sera possible d'ajouter un filigrane légal ou de conserver le filigrane légal existant.
  7. Cliquez sur Retranscoder.

Pendant la création des nouveaux rendus, la vidéo restera lisible en utilisant les rendus existants.
