Étapes

  1. Dans la navigation principale, cliquez sur Media.
  2. Dans le panneau de navigation latéral, cliquez sur le lien Créer une liste de lecteur.
  3. Saisissez un nom de liste de lecteur.
    4. {% if site.product_short == Cloud vidéo-fr/ %}
  4. Dans la section Type de liste de lecteur, cliquez sur Liste de lecteur intelligente.
    5. {% endif %} {% if site.product_short == Beacon-fr/ ou site.product_short == CorpTV-fr/ %}
  5. Dans la section Type de liste de lecteur, cliquez sur la liste de lecteur intelligente Video Cloud.
    6. {% endif %}
  6. Cliquez sur Créer une playlist.
  7. Sélectionnez un paramètre de balises.
  8. Définissez l'opérateur sur Faire correspondre n'importe quel.
  9. Entrez la ou les balises appropriées et appuyez sur Entrée.
  10. Cliquez sur Appliquer.
  11. Cliquez sur l'onglet Vue d'ensemble.
  12. Cochez Afficher la liste de lecteur dans la barre latérale pour que la liste de lecteur apparaisse dans la barre
  13. Fermez le panneau en cliquant sur le X.
  14. Cliquez sur le nom de la playlist que vous venez de créer dans le panneau de navigation latéral et vérifiez que les bonnes vidéos ont été ajoutées à la liste de lecteur.
Remarque: Pour plus de détails sur la création d'une liste de lecteur intelligente, consultez Création et gestion de listes de lecteur.
