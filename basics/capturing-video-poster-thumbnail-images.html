Étapes

  1. Dans la navigation principale, ouvrez le module Médias.
  2. Cliquez sur le titre d'une vidéo pour ouvrir la page des propriétés de la vidéo.
  3. Utilisez le lecteur de prévisualisation en haut de la page pour lire la vidéo.
  4. Mettez la vidéo en pause lorsque vous arrivez à l'image que vous souhaitez utiliser pour l'affiche/la vignette.
  5. Cliquez sur Capture.
  6. Sélectionnez cette option pour utiliser l'image comme affiche et/ou vignette.
  7. Cliquez sur Enregistrer.

Notez que le traitement peut prendre une minute et que les images apparaissent dans le module Médias.
