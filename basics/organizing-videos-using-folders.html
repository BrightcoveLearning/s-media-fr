Étapes

  1. Dans la navigation principale, ouvrez le module Médias.
  2. Dans le panneau de navigation latéral, cliquez sur le lien Créer un dossier.
  3. Saisissez un nom pour le dossier, puis appuyez sur Entrée.
  4. Pour ajouter des vidéos à un dossier, sélectionnez une ou plusieurs vidéos.
  5. Cliquez sur Plus Ë‡ > Déplacer vers le dossier.
  6. Cliquez sur un dossier, puis sur Déplacer.

Tous les dossiers apparaissent dans le menu de navigation de gauche. Le numéro qui suit le nom du dossier correspond au nombre de vidéos qu'il contient.

Pour supprimer des vidéos d'un dossier, cliquez sur un dossier, sélectionnez une ou plusieurs vidéos, puis cliquez sur Supprimer les vidéos Ë‡ > Supprimer les vidéos du dossier.

Suppression et changement de nom de dossiers

Les dossiers peuvent être renommés ou supprimés en procédant comme suit:

  1. Cliquez sur le dossier à renommer ou à supprimer.
  2. Cliquez sur la flèche vers le bas en regard du dossier, puis cliquez sur Renommer ou Supprimer.
Remarque: Pour plus de détails sur l'utilisation des dossiers, consultez la section Organisation des vidéos à l'aide de dossiers.
