Étapes

  1. Dans la navigation principale, cliquez sur Media.
  2. Dans le panneau de navigation latéral, cliquez sur le lien Créer une liste de lecteur.
  3. Saisissez un nom de liste de lecteur.
    4. {% if site.product_short == Cloud vidéo-fr/ %}
  4. Pour le type Playlist, cliquez sur Playlist manuelle.
    5. {% endif %} {% if site.product_short == Beacon-fr/ ou site.product_short == CorpTV-fr/ %}
  5. Pour le type de liste de lecteur, cliquez sur la liste de lecteur manuelle de Video Cloud.
    6. {% endif %}
  6. Cliquez sur Créer une playlist. Le panneau va se fermer puis rouvrir.
  7. Cochez Afficher la liste de lecteur dans la barre latérale pour que la liste de lecteur apparaisse dans la barre
  8. Cliquez sur X pour fermer le panneau.
  9. Pour ajouter des vidéos à la liste de lecteur, cliquez sur Toutes les vidéos dans le panneau de navigation latéral afin d'afficher toutes les vidéos du compte.
  10. Cochez la case en regard des vidéos que vous souhaitez ajouter à la liste de lecteur.
  11. Cliquez sur Plus Ë‡ > Ajouter à la liste de lecteur.
  12. Sélectionnez une liste de lecteur, puis cliquez sur Ajouter.

Pour réorganiser l'ordre des vidéos, sélectionnez la liste de lecteur, puis cliquez sur une ligne et faites-la glisser vers un nouvel emplacement dans la liste.

Remarque: Pour plus de détails sur la création d'une liste de lecteur manuelle, voir Création et gestion de listes de lecteur.
