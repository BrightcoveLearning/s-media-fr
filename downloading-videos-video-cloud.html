Les vidéos peuvent être téléchargées de différentes manières à partir d'un compte {{site.product_short}}. Les vidéos peuvent être téléchargées à l'aide des modules Médias ou Réseaux sociaux, en utilisant les API et un lien de téléchargement peut être ajouté à un lecteur permettant aux spectateurs de télécharger du contenu.

Téléchargement de vidéos à l'aide du module Médias

Vous pouvez télécharger un rendu d'une vidéo à l'aide du module Médias. Si l'icône du nuage à côté du nom de la vidéo dans le module Media est grise (), la vidéo a été ingérée à l'aide d'un profil d'acquisition hérité et les rendus MP4 et HLS peuvent être téléchargés dans le module Media. Une icône de nuage verte () indique que la vidéo a été ingérée à l'aide de la diffusion dynamique et si un profil d'acquisition qui crée des rendus MP4 a été sélectionné, les rendus MP4 peuvent être téléchargés.

Remarques:
  • La section Fichiers vidéo de la page des propriétés de la vidéo indique également quel type de profil d'ingestion a été utilisé.
  • Si vous souhaitez qu'un rendu 1080p soit disponible au téléchargement, vous devez créer un profil d'acquisition personnalisé et vous assurer qu'il inclut au moins l'un des formats associés suivants:
    • Progressif 3500
    • progressif 4000

  2. Dans le panneau de navigation, cliquez sur Toutes les vidéos.

  3. Cliquez sur le lien correspondant à la vidéo que vous souhaitez télécharger.
    4. {% if site.product_short == Beacon-fr/ ou site.product_short == CorpTV-fr/ %}
  4. Cliquez sur l'onglet Vidéo et pistes.

  5. Localisez la section Fichiers vidéo.
  6. Vérifiez le type de transcodage utilisé:
    • Diffusion dynamique (icône de nuage vert, exemple ci-dessous): cliquez avec le bouton droit sur un lien dans la colonne Dimensions pour le format associé à télécharger ; notez qu'un lien de téléchargement cliquable ne s'affichera que si des rendus MP4 ont été créés
    • Legacy (icône de nuage gris) - Cliquez avec le bouton droit sur le lien dans la colonne Format pour le format associé à télécharger
Remarque: Les rendus générés par Dynamic Delivery ont une durée de vie (TTL) de 6 heures. Les liens générés seront valides pendant 6 heures.

Il est important de comprendre que les URL des rendus et des images ne sont pas corrigées. Brightcove reconfigure le stockage des ressources multimédias de temps à autre et lorsque cela se produit, les URL de ressources spécifiques changent. Si vous utilisez des URL codées en dur vers ces ressources, les liens peuvent être rompus à un moment ou à un autre. Le meilleur moyen d'empêcher la rupture des liens vers les médias est de les récupérer à partir de {{site.product_short}} au moment de l'exécution à l'aide de l'API CMS.

Téléchargement de vidéos à l'aide du module Social

Le module Social permet de copier l'URL de la vidéo ou de télécharger un fichier source vidéo. Lors du téléchargement, le rendu MP4 de la plus haute qualité sera téléchargé.

Pour télécharger une vidéo à l'aide du module Social, procédez comme suit:

  1. Ouvrez Brightcove Social.
  2. Cliquez sur Vidéos dans le menu de navigation de gauche. Une liste des vidéos de votre compte {{site.product_short}} s'affichera.
  3. Cliquez sur la vidéo à télécharger, puis cliquez sur Télécharger.
    Remarque: S'il n'existe aucun rendu MP4, le lien de téléchargement sera désactivé.
    Remarque: Vous ne pouvez pas télécharger de vidéos pour lesquelles la gestion des droits numériques est activée (le verrou apparaît dans la miniature de la vidéo). Si vous sélectionnez une vidéo compatible DRM, le bouton Télécharger sera désactivé.
  4. Copiez l'URL de la vidéo ou cliquez sur le lien pour ouvrir la vidéo dans votre navigateur. Cliquez avec le bouton droit sur la vidéo pour enregistrer le fichier source localement.
  5. Cliquez sur Terminé.

Accès aux propriétés vidéo à l'aide de l'API CMS

L'API CMS peut être utilisée pour récupérer les métadonnées, y compris les URL des vidéos, pour n'importe quelle vidéo d'un compte {{site.product_short}}. Des compétences en développement sont requises pour utiliser l'API CMS. Pour plus d'informations sur l'accès aux propriétés vidéo à l'aide de l'API CMS, consultez la section Utilisation de l'API CMS: Récupérer le document du développeur Video Data.

Brightcove Learning Services a également créé un exemple d'application API CMS que vous pouvez utiliser pour obtenir des liens de téléchargement: Exemple d'API CMS: Liens de téléchargement.

Création d'un lien de téléchargement pour les fichiers vidéo

Si vous avez des compétences en développement, il est possible de placer un lien de téléchargement dans la barre de contrôle du lecteur pour donner aux spectateurs la possibilité de télécharger du contenu.

Vous pouvez également afficher un lien de téléchargement en superposition du lecteur ou en dessous du lecteur. Pour plus d'informations sur la procédure à suivre, y compris un exemple de code, voir Exemple de lecteur Brightcove: Téléchargez le plugin vidéo.
