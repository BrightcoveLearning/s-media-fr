Le module Upload utilise un processus d'acquisition dynamique pour récupérer et transcoder le contenu vidéo. Lorsque vous sélectionnez un fichier à télécharger, {{site.product_short}} utilise un processus basé sur l'extraction pour extraire le contenu d'un emplacement de stockage où il utilise ensuite le service de transcodage Zencoder pour transcoder le contenu.

Le module Upload prend en charge différentes méthodes de téléchargement de vidéos:

  • Drag and Drop - Cette méthode vous permet de faire glisser des fichiers de votre système de fichiers local vers la page du module Upload
  • Boîte de dialogue de sélection de fichiers: cette méthode ouvre une boîte de dialogue de sélection de fichiers dans laquelle vous pouvez parcourir et sélectionner les fichiers à
  • URL distante - Cette méthode vous permet de saisir une URL pour spécifier l'emplacement du fichier source

Cette rubrique se concentre sur la manière dont vous pouvez obtenir l'URL distante pour les services d'hébergement de fichiers cloud les plus populaires. L'utilisation de la méthode de téléchargement d'URL à distance est utile pour les fichiers plus volumineux qui peuvent avoir été modifiés/traités par des personnes extérieures à votre organisation. Plutôt que de copier un fichier volumineux sur votre machine locale, une URL distante vers le fichier peut être fournie et {{site.product_short}} peut extraire le fichier de l'emplacement. Dans tous les cas, vous devez vous assurer que l'URL du fichier est accessible au public, faute de quoi le téléchargement échouera.

Remarque: Pour plus de détails sur l'utilisation du module Upload, voir Chargement de vidéos à l'aide du module Upload.
Remarque: Avant de télécharger des vidéos, assurez-vous que le profil d'acquisition approprié a été sélectionné.

Google Drive

Pour obtenir l'URL d'une vidéo stockée sur Google Drive, procédez comme suit:

  1. Connectez-vous à votre compte Google Drive.
  2. Localisez la vidéo que vous souhaitez mettre en ligne et cliquez dessus avec le bouton droit.
  3. Cliquez sur le lien Partageable.
  4. Pour les autorisations, assurez-vous de sélectionner l'option afin que toute personne disposant du lien puisse accéder au fichier. Si l'autorisation est définie sur Restreint, le téléchargement échouera.
    Remarque: Si vous possédez un compte Google d'entreprise, vos seules options de partage peuvent être limitées ou le partage sur votre domaine. Si vous n'avez pas la possibilité de partager le fichier avec n'importe qui avec le lien, {{site.product_short}} ne pourra pas accéder au fichier et le téléchargement échouera.
  5. Cliquez sur Copier le lien. Malheureusement, cela ne génère pas de lien direct vers le fichier vidéo. Nous pouvons utiliser un autre site pour générer un lien direct vers le fichier vidéo à partir du lien de partage.
  6. Accédez à https://sites.google.com/site/gdocs2direct/home. D'autres sites sont également disponibles pour générer un lien direct vers les fichiers Google Drive.
  7. Collez votre URL de partage.
  8. Cliquez sur Créer un lien direct.

L'URL renvoyée dans le champ Lien de sortie est l'URL à utiliser dans le module Upload pour télécharger la vidéo.

Remarque: Vous ne pouvez pas utiliser d'URL YouTube car aucun lien direct vers des fichiers vidéo n'est généré.
Remarque: Les téléchargements ne fonctionneront pas si un avertissement de taille s'affiche pour Google Drive (généralement pour les fichiers > 100 Mo). Même après avoir modifié l'URL, elle sera redirigée vers une page d'erreur de taille et le téléchargement échouera.

Dropbox

Pour obtenir l'URL d'une vidéo stockée sur Dropbox, procédez comme suit:

  1. Connectez-vous à votre compte Dropbox.
  2. Localisez la vidéo que vous souhaitez mettre en ligne et cliquez sur le lien Partager.
  3. Cliquez sur Créer un lien pour créer un lien vers le fichier.
  4. Cliquez sur Copier le lien pour copier le lien dans le fichier.

Remarques:

  • L'URL ne doit pas contenir d'espaces. S'il y a des espaces dans le nom de fichier ou le chemin d'accès, l'encodage remplace généralement l'espace par %20. Renommez tous les dossiers ou fichiers, en remplaçant les espaces par des traits de soulignement (_) lors de l'utilisation de ces liens pour le téléchargement.
  • L'URL du fichier peut se terminer par ? dl=0. Pour forcer le navigateur à télécharger le contenu du lien, vous devez remplacer cette partie de l'URL par ? dl=1.

Box

Pour obtenir l'URL d'une vidéo stockée sur Box, procédez comme suit:

Remarque: La possibilité d'obtenir un lien direct vers un fichier est une fonctionnalité premium de Box, qui n'est pas disponible sur les comptes personnels gratuits.
  1. Connectez-vous à votre compte Box.
  2. Localisez la vidéo que vous souhaitez mettre en ligne et cliquez sur le lien Partager.
  3. Cliquez sur le curseur Créer un lien partagé pour créer un lien vers le fichier. Cela créera un lien d'aperçu vers le fichier.
  4. Cliquez sur Paramètres des liens. Vous devez assigner un lien de téléchargement direct au fichier.
  5. Cliquez sur Copier pour copier le lien dans le presse-papiers.

Autres services

Il ne devrait y avoir aucun problème lors de l'utilisation des URL pour les fichiers vidéo hébergés sur d'autres plateformes cloud (par exemple Amazon S3) tant que l'URL est une URL publique.

Remarque: Lors de l'acquisition de fichiers vidéo depuis Amazon S3, il est possible de créer une stratégie de compartiment afin que les vidéos qui ne sont pas accessibles à tous soient accessibles à partir de {{site.product_short}}. Pour plus d'informations, consultez le document destiné aux développeurs Using Dynamic Ingest with S3.
