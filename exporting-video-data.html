Le module Médias permet d'exporter les métadonnées vidéo de toutes vos vidéos ou d'un sous-ensemble de vidéos. Lorsque vous utilisez cette fonctionnalité, tenez compte des points suivants:

  • Brightcove recommande d'exporter moins de 15 000 vidéos à la fois. Si vous devez exporter plus de 15 000 vidéos, utilisez la fonction de recherche pour renvoyer un sous-ensemble de vidéos et exporter plusieurs lots-fr/ à la fois.
  • Les URL des rendus et des images ne sont pas corrigées. Brightcove reconfigure le stockage des ressources multimédias de temps à autre et lorsque cela se produit, les URL de ressources spécifiques changent. Si vous utilisez des URL codées en dur vers ces ressources, les liens peuvent être rompus à un moment ou à un autre. Le meilleur moyen d'empêcher la rupture des liens vers les médias est de les récupérer à partir de {{site.product_short}} au moment de l'exécution à l'aide de l'API CMS.
  • Les caractères codés sur deux octets tels que le coréen et le japonais ne s'affichent pas correctement s'ils sont ouverts dans Excel. Cela est dû au fait que le fichier .csv généré est codé en UTF-8 et qu'Excel ne le prend pas en charge correctement. Pour résoudre ce problème, le fichier .csv doit être réencodé par ANSI, puis il s'affichera correctement dans Excel.

Pour exporter les métadonnées vidéo, procédez comme suit.

  1. Ouvrez le module Médias.
  2. Si vous souhaitez exporter des données pour des vidéos spécifiques, cochez la case en regard des vidéos.
  3. Cliquez sur Exporter en haut de la page.
  4. Sélectionnez les vidéos à exporter:
    • Toutes les vidéos de cette liste - Exporte les métadonnées vidéo de toutes les vidéos du compte
    • x vidéos sélectionnées - Exporte les métadonnées des vidéos uniquement pour les vidéos sélectionnées
  5. Cochez la case Inclure les URL actuelles pour les rendus vidéo pour inclure les URL de rendu vidéo. Cocher cette option avec un grand nombre de vidéos sélectionnées augmentera le temps nécessaire à la génération du fichier d'exportation.
    Remarque: Lorsque cette option est sélectionnée, les métadonnées vidéo d'un maximum de 1 000 vidéos peuvent être exportées.
  6. Cliquez sur Exporter les données. L'exportation des données va commencer.
  7. Lorsque le fichier d'exportation est prêt, cliquez sur Télécharger le fichier pour télécharger le fichier de données vidéo sur votre ordinateur local.

Le fichier .csv sera nommé avec le numéro de compte {{site.product_short}}. Un tableur tiers peut être utilisé pour afficher les données vidéo exportées.
