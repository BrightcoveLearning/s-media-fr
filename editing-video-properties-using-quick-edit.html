La fonction Quick Edit permet de modifier facilement les propriétés des vidéos de plusieurs vidéos. Cliquez sur Quick Edit pour ouvrir le panneau Quick Edit sur le côté droit du module Media.

Le panneau Quick Edit affiche un sous-ensemble des propriétés de la vidéo:

  • Nom
  • Brève description
  • Description longue
  • Tags à ajouter
  • Balises à supprimer
  • URL du lien associé
  • Texte du lien associé
  • Publicité
  • Clés publicitaires
  • Planification des dates de début et de fin
  • Géofiltrage
  • Champs personnalisés
    • {% if site.product_short == Cloud vidéo-fr/ %}
  • Restrictions de
    • {% endif %}
Remarque: Pour une description de toutes ces propriétés, consultez la rubrique Modification des propriétés vidéo à l'aide du module multimédia.

Utilisez le panneau Quick Edit pour attribuer des valeurs. Lorsque vous avez terminé, cliquez sur Enregistrer pour enregistrer les modifications ou sur Annuler pour quitter.

Remarques relatives à l'utilisation de Quick

  • Lorsqu'au moins une vidéo possède des valeurs attribuées à une propriété, Edit Mixed Value (Modifier la valeur mixte) s'affiche
  • Un astérisque (*) suivant le nom de la propriété indique une propriété requise
  • Pour annuler les modifications apportées à une propriété, cliquez sur Restaurer
