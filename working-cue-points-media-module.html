Un point de repère est un marqueur à un moment précis de la durée d'une vidéo. Vous pouvez utiliser des points de repère pour déclencher des publicités mid-roll (c'est-à-dire des publicités qui apparaissent pendant la lecteur du contenu vidéo, comme une pause publicitaire) ou pour séparer des chapitres ou des scènes dans une vidéo de longue durée.

Il existe deux types de points de repère, mais les deux fournissent simplement une valeur de chaîne à la propriété type de l'objet point de repère. La valeur appropriée n'est utile que si elle est utilisée dans du JavaScript personnalisé lors du traitement des informations de point de repère. 

  • Ad - Attribue la valeur de chaîne de AD à la propriété type
  • Code - Attribue la valeur de chaîne de CODE à la propriété type

Les points de repère peuvent être ajoutés aux vidéos à l'aide du module Médias ou de l'API Catalog. Pour plus d'informations sur l'ajout de points de repère à l'aide de l'API Catalog, consultez le document du développeur Implementing Cue Points Programmatically

Remarques:
  • Les points de repère ne peuvent pas être ajoutés aux vidéos de ressources distantes en
  • Les points de repère doivent être séparés d'au moins une seconde.

    • Ajout de points de repère

    Pour ajouter des points de repère à une vidéo, procédez comme suit:

    1. Ouvrez le module Médias.
    2. Cliquez sur le lien du titre de la vidéo pour ouvrir la page des propriétés de la vidéo.
    3. Utilisez le lecteur de prévisualisation pour lire la vidéo. Mettez la vidéo en pause à l'endroit où vous souhaitez ajouter un point de repère.
    4. Cliquez sur Ajouter un point de repère.
    5. Si nécessaire, ajustez les valeurs du code temporel (hh:mm:ss) pour le point de repère. Notez que le champ secondes prend en charge les millisecondes après la virgule décimale.
    6. Sélectionnez le type de point de repère, puis ajoutez les métadonnées appropriées.
      • ad - Une demande d'annonce sera effectuée. Lorsque vous utilisez des points de repère publicitaires, le lecteur doit être configuré pour la publicité.
      • code - Les points de repère seront gérés par programmation. Consultez le document du développeur Implémentation des points de repère par programmation pour plus d'informations.
    7. Entrez un nom pour le point de repère. La valeur du nom est accessible par programme.
    8. Entrez des métadonnées (pour les points de repère de code) ou des paires clé/valeur (pour les points de repère publicitaires).
      • Métadonnées: métadonnées facultatives stockées avec le point de repère. Les métadonnées sont accessibles par programme. Par exemple, vous pouvez utiliser un point de repère de code et transmettre des métadonnées lorsque vous souhaitez afficher différents appels à l'action pendant la lecteur de la vidéo. Consultez l'exemple de lecteur Brightcove: Les points de repère affichent le CTA par exemple.
      • Paires clé/valeur: paires clé/valeur pouvant être transmises à une demande d'annonce. Une paire clé-valeur est un ensemble de deux éléments de données liés: une clé, qui est un identifiant unique pour certains éléments de données, et la valeur, qui est soit les données identifiées, soit un pointeur vers l'emplacement de ces données. Les paires clé/valeur peuvent être utilisées pour fournir des publicités plus ciblées. Tout ce qui est saisi ici sera copié dans la balise d'annonce où la macro {mediainfo.ad_keys} est insérée. Pour plus d'informations, consultez Publicité avec le plugin IMA3.

    Une fois les points de repère ajoutés à une vidéo, au moment de l'exécution, les points de repère sont convertis en éléments de repère HTML5 WebVTT. Lorsque cette conversion a lieu, certaines des données de point de repère, telles que le type et l'heure du point de repère, sont converties dans la piste de texte. La valeur temporelle du point de repère sera mappée dans les propriétés StartTime et EndTime du signal WebVTT. Consultez ce document pour plus d'informations sur l'interface VTTCue.

    Édition des points de repère

    Pour modifier les points de repère d'une vidéo, procédez comme suit:

    1. Ouvrez le module Médias.
    2. Cliquez sur le lien du titre de la vidéo pour ouvrir la page des propriétés de la vidéo.
    3. Cliquez sur le point de repère pour le modifier dans la chronologie.
      Remarque: Une fois en mode édition, les boutons Précédent et Suivant peuvent être utilisés pour accéder à d'autres points de repère.
    4. Modifiez les propriétés du point de repère, puis cliquez sur Enregistrer.
    Remarque: Il n'est pas possible d'avoir plusieurs points de repère avec le même code temporel.

    Supprimer des points de repère

    Pour supprimer des points de repère, procédez comme suit:

    1. Ouvrez le module Médias.
    2. Cliquez sur le lien du titre de la vidéo pour ouvrir la page des propriétés de la vidéo.
    3. Cliquez sur le point de repère à supprimer dans la chronologie.
    4. Cliquez sur l'icône Supprimer () en regard des propriétés du point de repère.
    5. Confirmez la suppression.
