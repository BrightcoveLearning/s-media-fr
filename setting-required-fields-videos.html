De nombreux éditeurs utilisent les métadonnées vidéo (noms, balises, identifiants de référence, champs personnalisés, etc.) comme outils essentiels pour suivre et monétiser leur contenu. Pour que les métadonnées vidéo soient efficaces, vous devez vous assurer que chacun des champs de métadonnées dont vous dépendez est bien présent. La fonctionnalité de sécurisation des métadonnées vous permet de désigner les champs de métadonnées vidéo importants pour vos besoins professionnels, puis d'exiger des utilisateurs qu'ils remplissent les champs de métadonnées requis pour les nouvelles vidéos avant que celles-ci ne soient actives.

Désignation des champs de métadonnées selon les besoins

Les champs de métadonnées vidéo obligatoires sont configurés dans le cadre des paramètres du compte {{site.product_short}}. Les champs de métadonnées standard et personnalisés peuvent être marqués selon les besoins. Pour configurer les champs obligatoires:

  1. Ouvrez le module Admin.
  2. Cliquez sur le lien Video Fields.
  3. Cliquez sur le nom du champ que vous souhaitez définir selon vos besoins.
  4. Cochez la case Rendre ce champ obligatoire.
  5. Cliquez sur Enregistrer le champ.

À tout moment, vous pouvez décocher la case Rendre ce champ obligatoire pour n'importe quel champ si vous ne souhaitez plus que ce champ soit obligatoire. À partir de ce moment, {{site.product_short}} ne vérifiera plus si ce champ a été rempli.

Quels champs de métadonnées peuvent être des champs obligatoires ?

Le champ du nom de la vidéo est toujours obligatoire. Tout champ de métadonnées personnalisé peut être un champ obligatoire. En outre, les champs de métadonnées standard suivants peuvent être requis:

  • Publicité
  • Géo-filtrage activé
  • Brève description
  • Description longue
  • ID de référence
  • Texte du lien associé
  • URL du lien associé
  • Date de début de planification
  • Date de fin de planification
  • Balises

Vidéos sans métadonnées requises

Si une vidéo de votre médiathèque ne contient pas toutes les métadonnées requises, alors:

  • S'il est déjà actif, il reste actif et jouable. Cela signifie que lorsque vous spécifiez les champs de métadonnées requis pour votre compte, votre vidéothèque existante ne deviendra pas soudainement illisible tant que vous n'aurez pas mis à jour les métadonnées requises.
  • S'il est déjà actif et que vous le modifiez dans le module Médias, vous ne pouvez pas enregistrer vos modifications sans fournir toutes les métadonnées requises dans la section que vous modifiez.
  • S'il est inactif, vous ne pouvez pas le rendre actif sans fournir toutes les métadonnées requises

Téléchargement de vidéos sans métadonnées requises

Par défaut, lorsqu'une vidéo est téléchargée à l'aide du module Upload, l'état de la vidéo est défini sur Actif, sauf si des champs obligatoires sont manquants. S'il manque des champs obligatoires, un message s'affiche avec les champs obligatoires surlignés en rouge. Les vidéos dont les champs obligatoires sont manquants peuvent toujours être téléchargées, mais elles resteront inactives jusqu'à ce que tous les champs obligatoires aient des valeurs.

Montage de vidéos

Si une vidéo est inactive, vous pouvez modifier et enregistrer librement ses métadonnées dans le module Médias sans fournir toutes les métadonnées requises. En outre, la vidéo ne peut pas être activée tant que toutes les métadonnées requises n'ont pas été saisies.

Si la vidéo est déjà active, vous ne pouvez enregistrer aucune modification dans la section que vous modifiez, sauf si vous fournissez toutes les métadonnées requises dans cette section ou si vous désactivez la vidéo.

Activation des vidéos

Si vous souhaitez activer une vidéo dans le module Médias, elle doit disposer de toutes les métadonnées requises. Si vous essayez d'activer une vidéo dont les métadonnées requises sont manquantes, une erreur s'affiche.

La barre d'état en bas indique si vos vidéos peuvent être activées. Le bouton Activer reste désactivé tant que vous n'avez pas fourni toutes les métadonnées requises pour les vidéos.

Conseil sur le workflow

Vous voulez retrouver toutes vos vidéos inactives ? Dans le module Médias, vous pouvez trier toutes vos vidéos par statut. Il suffit de cliquer dans l'en-tête de la colonne Statut. Ensuite, vous pouvez modifier chacune des vidéos inactives pour voir les métadonnées requises qui peuvent leur manquer.

Limitations

Les champs obligatoires ne sont appliqués que dans le module Médias. Vous ne pouvez pas activer une vidéo inactive dans le module Médias tant que tous les champs de métadonnées requis ne sont pas remplis. Cependant, une vidéo sans champs de métadonnées obligatoires peut toujours devenir active des manières suivantes:

  • La vidéo a été créée dans votre compte et activée avant que vous ne spécifiiez les champs obligatoires pour votre compte
  • La vidéo a été téléchargée à l'aide de l'API Dynamic Ingest

En outre, {{site.product_short}} valide le type mais ne valide pas les valeurs des métadonnées requises. Tant que la valeur fournie est du type correct répertorié pour le champ dans les paramètres du compte (par exemple, texte, URL ou date), les utilisateurs peuvent entrer les valeurs de leur choix.
