Les dossiers peuvent être créés dans le module Médias et peuvent être utilisés pour organiser des vidéos. Vous pouvez choisir d'organiser les vidéos par sujet, auteur, produit, région, etc. Le fait d'avoir des vidéos dans des dossiers facilite leur gestion dans un compte contenant des centaines ou des milliers de vidéos. Les dossiers sont affichés dans le panneau de navigation latéral. Le nombre de vidéos de chaque dossier s'affiche en regard du nom du dossier. Cliquez sur le bouton de navigation latéral pour masquer ou afficher le panneau de navigation latéral.

Remarques:
  • Les dossiers imbriqués ne sont pas pris en charge.
  • Un maximum de 100 vidéos peuvent être déplacées dans un dossier en même temps.
  • Chaque compte peut contenir un maximum de 1 000 dossiers.

Création de dossiers

Pour créer un nouveau dossier, procédez comme suit:

  1. Ouvrez le module Médias.
  2. Cliquez sur Créer un dossier dans le panneau de navigation latéral.
  3. Entrez un nom pour le dossier en suivant les instructions suivantes. Appuyez sur Entrée.
    • Le nom est limité à 100 caractères
    • Les noms de dossiers doivent être uniques au sein d'un compte
    • Caractère illégal:/
Remarque: Il est possible de créer un maximum de 1 000 dossiers.

Pour supprimer rapidement le dossier, cliquez sur le X après le nom du dossier.

Ajout de vidéos aux dossiers

Il existe plusieurs manières d'ajouter des vidéos à un dossier. Vous pouvez:

  • Ajouter plusieurs vidéos à un dossier
  • Modifier les propriétés de la vidéo pour attribuer un dossier
Remarque: Il n'y a pas de limite au nombre de vidéos pouvant se trouver dans un dossier.

Ajout de plusieurs vidéos à un dossier

Vous pouvez ajouter plusieurs vidéos à la fois à un dossier en suivant ces étapes.

  1. Ouvrez le module Médias.
  2. Cliquez sur Toutes les vidéos dans le panneau de navigation latéral pour afficher toutes les vidéos.
    Remarque: Le chiffre qui suit le nom du dossier indique le nombre de vidéos qu'il contient.
  3. Sélectionnez les vidéos à ajouter à un dossier en cochant la case en regard de la miniature de la vidéo. Cliquez sur Plus > Déplacer vers le dossier.
  4. Sélectionnez un dossier et cliquez sur Déplacer

Modification des propriétés de la vidéo pour attribuer un dossier

Les vidéos peuvent être ajoutées à un dossier en modifiant les propriétés de la vidéo et en attribuant un dossier.

  1. Ouvrez le module Médias.
  2. Cliquez sur le lien de la vidéo.
  3. Dans la section Catégorisation, sélectionnez un dossier. Vous pouvez également créer un nouveau dossier en cliquant sur + Créer un dossier.

Visionnage de vidéos dans un dossier

Pour afficher les vidéos d'un dossier, cliquez sur le nom d'un dossier.

Suppression de vidéos de dossiers

Vous pouvez supprimer plusieurs vidéos d'un dossier ou supprimer une seule vidéo d'un dossier en modifiant les propriétés de la vidéo.

Suppression de plusieurs vidéos d'un dossier

Vous pouvez supprimer plusieurs vidéos d'un dossier en suivant ces étapes.

  1. Ouvrez les médias
  2. Cliquez sur le lien du dossier.
  3. Cochez les cases en regard des vidéos à supprimer et cliquez sur Supprimer les vidéos > Supprimer les vidéos du dossier.

Modification des propriétés de la vidéo

Les vidéos peuvent être supprimées d'un dossier en modifiant les propriétés de la vidéo et en supprimant le dossier dans le panneau Informations sur la vidéo.

  1. Ouvrez les médias
  2. Cliquez sur le lien du dossier.
  3. Dans la section Catégorisation, définissez le dossier sur ---.

Notez que cette méthode peut également être utilisée pour sélectionner un autre dossier dans lequel déplacer la vidéo.

Suppression et changement de nom de dossiers

Les dossiers peuvent être renommés ou supprimés en procédant comme suit:

  1. Cliquez sur le dossier à renommer ou à supprimer.
  2. Cliquez sur la flèche vers le bas en regard du dossier, puis cliquez sur Renommer ou Supprimer.
